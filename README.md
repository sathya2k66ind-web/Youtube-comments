## 💡 Features
- Extract YouTube comments via API
- Sentiment analysis: Positive, Neutral, Negative
- Insights & statistics from comments
- Easy-to-use web interface (Flask backend)

## 🛠 Tech Stack
- **Backend:** Python, Flask  
- **ML / NLP:** Transformers, HuggingFace, TextBlob, Scikit-learn  
- **Frontend:** HTML, CSS, Bootstrap  
- **Deployment:** Render (free tier)
