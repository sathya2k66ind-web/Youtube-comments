## 🔮 Core Features
### 🧠 The Brain (Backend)
- **API Extraction:** Seamlessly pulls raw comment data streams via the YouTube API.
- **Deep Sentiment Analysis:** Classifies data into `[POSITIVE]`, `[NEUTRAL]`, and `[NEGATIVE]`.
- **Toxicity Radar:** Automatically flags highly aggressive or toxic comments with a blinking warning.

### 💿 The Aesthetic (Frontend)
- **100% Custom Y2K UI:** Completely framework-free. Built with advanced CSS keyframes, 3D transforms, and custom styling.
- **CRT Monitor Effects:** Authentic retro feel with animated screen flicker and RGB scanlines.
- **Cyber-Grid 3D Floor:** An infinitely scrolling, perspective-shifted neon grid background.
- **Glitch Physics:** Hovering over critical buttons triggers complex CSS clip-path RGB glitch animations.
- **Retro OS Window:** The app lives inside a classic Win98-style interface, complete with inset/outset 3D bevels and Chrome WordArt.
- **Terminal Data Visualization:** Chart.js has been completely re-themed with neon lime/cyan grids and `VT323` hacker fonts.

---

## 🛠 Tech Stack
- **Backend Core:** Python, Flask 
- **Machine Learning:** HuggingFace Transformers, TextBlob, Scikit-learn
- **Frontend Engine:** Pure HTML5, highly advanced Vanilla CSS3, Chart.js 
- **Deployment:** Render (Cloud Hosting)

*(Note: Previous versions used Bootstrap, but it was entirely stripped out to build this 100% bespoke Y2K aesthetic from scratch.)*
